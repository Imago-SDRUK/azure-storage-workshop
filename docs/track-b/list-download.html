Step 3 — List & download

You’re connected (from Authenticate, you have a container client). Now let’s see what’s in the container and pull files down.

List the files (blobs)

# List every blob in the container.
print("Files in the container:")
for blob in container.list_blobs():
    size_kb = (blob.size or 0) / 1024
    print(f"  {blob.name}  ({size_kb:,.1f} KB)")

list_blobs() returns each blob’s name, size, last_modified, and more.

Tip

If the container has a lot of files, filter by a name prefix instead of listing everything — this is the equivalent of opening one “folder”:

for blob in container.list_blobs(name_starts_with="reports/"):
    print(blob.name)

Download one file

from pathlib import Path

BLOB_NAME = "PLACEHOLDER-EXAMPLE-FILE"   # e.g. "sample-data.csv"
dest = Path("downloads") / BLOB_NAME
dest.parent.mkdir(parents=True, exist_ok=True)

blob_client = container.get_blob_client(BLOB_NAME)

with open(dest, "wb") as f:
    download = blob_client.download_blob()
    download.readinto(f)

print(f"Downloaded {BLOB_NAME} -> {dest}")

download_blob().readinto(f) streams the file straight to disk, so it works fine for large files without loading them all into memory.

Read a file’s contents directly (no saving)

Sometimes you just want the data in memory — e.g. load a CSV into pandas:

data = container.get_blob_client("PLACEHOLDER-EXAMPLE-FILE").download_blob().readall()
print(f"Read {len(data):,} bytes")

Download everything in the container

from pathlib import Path

out_dir = Path("downloads")

for blob in container.list_blobs():
    target = out_dir / blob.name
    target.parent.mkdir(parents=True, exist_ok=True)   # preserve "folder" structure
    with open(target, "wb") as f:
        container.get_blob_client(blob.name).download_blob().readinto(f)
    print(f"Downloaded {blob.name}")

A note on read-only

Everything above only reads. If you try a write or delete — for example container.upload_blob(...) or blob_client.delete_blob() — it will fail with a permissions error (403), because your SAS URL doesn’t grant those rights. That’s the intended safety net: you can’t change or remove our data.
